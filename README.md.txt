# 🎨 Викторина: История искусства России (ArtQuiz)

Интерактивное приложение-викторина, разработанное на Python. Проект включает в себя вопросы по истории русской живописи и архитектуры, систему подсчета очков и смену визуальных тем.

## 🚀 Особенности проекта
* **Графический интерфейс:** Реализован на библиотеке `Tkinter`.
* **Динамический контент:** Поддержка загрузки изображений для вопросов (форматы .jpg, .png).
* **Стабильность:** Внедрен алгоритм принудительной очистки графических контейнеров для предотвращения дублирования элементов.
* **Кастомизация:** Поддержка цветовых тем (светлая/темная/акцентная).

## 🛠 Технологический стек
* **Язык:** Python 3.x
* **Библиотеки:** `tkinter` (GUI), `Pillow` (обработка изображений), `random`, `json`.
* **Сборка:** `PyInstaller` (автономный .exe файл).

## 📂 Структура репозитория
* `RussianArtQuiz.pyw` — исходный код приложения.
* `icons/` — ресурсы (иконка приложения и изображения к вопросам).
* `RussianArtQuiz.exe` — готовая сборка для Windows.
