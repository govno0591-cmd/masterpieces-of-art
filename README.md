# 🎨 Шедевры искусства

Интерактивная викторина на Python, посвященная великим произведениям русского изобразительного искусства. Угадывайте названия картин и авторов, погружаясь в мир классической живописи.

---

### 📥 Скачать и запустить
Вы можете запустить игру двумя способами:

#### 1. Готовая версия (для всех)
Просто скачайте архив, распакуйте его и запустите исполняемый файл. Python не требуется.
👉 **[Скачать ArtQuiz v1.0 (.zip)](https://github.com/govno0591-cmd/masterpieces-of-art/releases/latest/download/ArtQuiz_1.0.zip)**

#### 2. Запуск из кода (для разработчиков)
Если у вас установлен Python, вы можете запустить проект напрямую:
1. Скачайте репозиторий.
2. Установите зависимости (если есть) из `requirements.txt`.
3. Запустите основной файл:
   ```bash
   python RussianArtQuiz.pyw
